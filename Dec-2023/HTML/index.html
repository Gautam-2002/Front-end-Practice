This page explains basics for HTML

HTML

Bookish People defination - HTML stands for Hyper Text Markup Language.

Actual Meaning of HTML - HTML is a markup language used to define stucture of web page.

HTML elements tell the browser how to display the content.

Now what is a element ?

HTML element is defined by a start tag, some content, and an end tag.

HTML elements are the building blocks of HTML pages.

HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML is not a programming language, it is a markup language. A markup language is a set of markup tags. HTML uses markup tags to describe web pages.

what is markup language ?

Markup language is a computer language that uses tags to define elements within a document. It is human readable language.

what is tag ?

Tag is a keyword surrounded by angle brackets. HTML tags are keywords used in HTML to display web-page on browser.

HTML tags are composed of three things: an opening tag, content and ending tag. Some HTML tags are self closing.
